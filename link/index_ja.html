Link Indicator

What's this?

Last modified : 2021/06/23 (Wed)

Link Indicatorはアンカーに付けられたtarget属性の判別とページ内リンクを視覚的に明示する仕組みを提供する補助スクリプトです。

  1. 機能
  2. 設置方法
  3. カスタマイズ

機能

概要へ

  • target="_blank"など外部ウィンドウを開くリンクを明示します（このサンプルでは OFF にしています）。
  • ページ内リンク[*1]を明示します。

[*1] ページ内ナビ以外にもこのような注釈にそれなりに有効です。

設置方法

概要へ

Link Indicatorの導入は簡単です。

  1. htmlヘッダにlink_indicator.jsを追記します。

カスタマイズ

概要へ

"link_indicator.js"の末尾に

addEvent(window,"load",function() {
  var indicator = new LinkIndicator({
    linkToBlank:'addwin.png',
    pointerImg:'pointer.png',
    borderWidth:'2px'
  });
});

のようにLink Indicatorの設定があります。

linkToBlankとpointerImgは画像のパスを設定します。画像を変更したり、画像を設置するディレクトリを変更した場合はここで画像のurlを指定してください。

デフォルトでは相対パスで指定され、"link_indicator.js"とそれを読み込むhtmlファイルと画像が全て同じディレクトリに置かれることが前提となっています。上記画像設定は絶対パスで指定した方が無難でしょう。

borderWidthでページ内リンクを明示するためのラインの太さを指定します。これを'0'に設定すると、ラインは引かれません。
