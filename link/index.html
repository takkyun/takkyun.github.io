Link Indicator

What's this?

Last modified : 2021/06/23 (Wed)

Link Indicator displays visually links within a page and links which have target attribute.

  1. Feature
  2. How to use
  3. Customize

Feature

overview

  • Displaying links which have target attribute like target="_blank" (It's turned off in this sample).
  • Displaying links within a page[*1].

[*1] It's effective for footnotes like this.

How to use

overview

It is very easy to use Link Indicator on your page.

  1. Just include link_indicator.js in the <head> of the content.

Customize

overview

At the end of "link_indicator.js",

addEvent(window,"load",function() {
  var indicator = new LinkIndicator({
    linkToBlank:'addwin.png',
    pointerImg:'pointer.png',
    borderWidth:'2px'
  });
});

there are settings for Link Indicator.

linkToBlank and pointerImg indicates paths for images. So you can change these when the images are changed or put on the different directory.

The default settings are set like above on the premise that html files, "link_indicator.js", and all images are put on the same directory. So I recommend to change it as absolute path, such as "https://takkyun.github.io/assets/images/addwin.png".

borderWidth indicates width of border to point links within a page. If you set as '0', a line is not drawn.
